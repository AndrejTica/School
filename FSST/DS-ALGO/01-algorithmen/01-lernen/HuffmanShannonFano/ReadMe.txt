Huffman/Shannon-Fano v1.1
=========================
Beschreibung:
  Huffman/Shannon-Fano ist ein in Java geschriebenes Lernprogramm für die binäre
  Huffman- und Shannon-Fano-Codierung.

Sprachausgabe:
  Die Sprachausgabe setzt Microsoft Windows XP oder Vista voraus, sowie die mit-
  gelieferte Datei say.exe. Sie lässt sich über den Menüpunkt Audio/Sprachausgabe
  aktivieren und verwendet die im System eingestellte Standardstimme. Es wird dringend
  empfohlen, eine deutschsprachige Stimme zu installieren. Mehr Informationen  dazu
  befinden sich auf der unten angegebenen Homepage.

Programmstart:
  Das Programm wird als ausführbare JAR-Datei veröffentlicht. Es lässt sich unter
  Windows mit einem Doppelklick auf "HuffmanShannonFano.jar" starten, sofern eine
  Java-Runtime der Version 5.0 (oder höher) installiert ist. Die Java-Runtime kann von 
  http://java.sun.com/ heruntergeladen werden.

Lizenz/Haftung:
  Dieses Programm ist Freeware und darf kostenlos genutzt und weitergegeben werden.
  Die Verwendung geschieht auf eigenes Risiko.
