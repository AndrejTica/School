Aufgabenstellungen

1. Erstellen Sie C-Programme zu den folgenden Aufgaben/Struktogrammen aus 
--------------------------------------------------------------------------------
01-grundlagen-struktogramme/01-lernen/01-grundlagen-struktogramme.odt
    Beispiel: flaeche
    Beispiel: tausche
    Beispiel: maxi
    Beispiel: summe1
    Beispiel: summe2
    Beispiel: pow
    Beispiel: teiler1
    Beispiel: teiler2

2. Erstellen Sie C-Programme zu den folgenden Aufgaben/Struktogrammen aus
--------------------------------------------------------------------------------
01-grundlagen-struktogramme/02-ueben/01-ue-grundlagen-struktogramme.odt
	Übung: Rechnen
	Übung: Division
	Übung: Summe2
	Übung: BMI
	Übung: Spiel21
	Übung: PerfekteZahl
	Übung: Primzahl


Hinweis:
	gcc hello.c -lm -o hello.exe
	
Hinweis: Wenn Mathematik Funktionen verwendet werden
	gcc hello.c -lm -o hello.exe

Hinweise 
Achten Sie 
	1. kompilierbare Quelltexte (keine Binaries!!!)  abgeben
	2. auf einen übersichtliche Formatierung und 
	3. auf eine sinnvolle Kommentierung des Quelltexts, 
	4. auf die Behandlung von Fehleingaben (falsche Argumente, usw.), 
	5. dass im Fehlerfall (Falscheingaben) Fehlermeldungen/Fehlercodes ausgegeben werden, 
	6. auf den geforderten Umfang der gestellten Aufgaben, und 
	7. dass die Programme ausführlich getestet sind, bevor Sie sie abgeben.
