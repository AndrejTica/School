# GPG (GNU Privacy Guard)
------------------------------------

<http://www.online-tutorials.net/security/gnupg-gpg-tutorial/tutorials-t-69-124.html>


## install
apt-get install gnupg
gpg --help


## Keys erzeugen
gpg --gen-key


## Widerrufs-Zertifikat erzeugen
(umask 077 && gpg --gen-revoke <USERID> > gpg-revoke.txt)



## Public-Key exportieren/importieren
gpg --export -a Anton Hofmann > public_key_Ahofmann.key 
gpg --import public_key_Ahofmann.key



## Public-KEYS anzeigen
gpg --list-public-keys
gpg --list-keys

## Private-Key anzeigen
gpg --list-secret-keys
